:doctitle: Welcome to the TED API Portal

TED API offers a comprehensive suite of APIs designed to streamline and enhance procurement processes. These APIs align with TED applications to support various procurement activities efficiently. Starting from version 3, TED API consolidates its services under a unified gateway, ensuring seamless integration and functionality.

== Available APIs and Their Purposes

TED APIs are made to address specific needs within the procurement ecosystem. Here is a brief overview of each API and its purpose:

=== Developer Operations API

Provides developers with auxiliary features to ensure smooth integration and compatibility with TED APIs, such as programmatic renewanl of API keys.

=== Conversion API

Enables the transition from legacy TED XML notices to modern eForms for users and systems alike.


=== Publication API

Automates the management and submission of procurement notices, integrating seamlessly with the eNotices2 system for efficient notice handling.

=== Validation API
Ensures compliance with EU standards by validating procurement notices for errors and potential legal issues before and upon submission.

=== Visualisation API

Transforms procurement notices into user-friendly formats like PDFs or HTML, supporting both public dissemination and internal use.

=== Search API

Provides access to published procurement notices, enabling data retrieval and analysis through expert queries and bulk downloads.
